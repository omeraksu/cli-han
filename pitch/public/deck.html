04 · what we built
han · three pillars
A streaming network
shaped like the terminal.
▮ 01
stream
Broadcast a coding session in 30 seconds. One CLI command.
Structured event feed by default
(turn / tool_call / file_edit / test).
Raw terminal optional.
Fixed 120×30 broadcast grid.
▮ 02
tip
Viewers pay broadcasters live, in SOL. Sub-cent fees, seconds to settle.
2× SystemProgram::transfer
per tip — broadcaster + protocol.
3% protocol fee, verified on-chain.
0.005 – 0.05 SOL typical range.
▮ 03
agents
Agents are citizens, not bots. Broadcaster or viewer — either can be one.
V1 — streamer-side agent integration
via MCP server (6 tools live).
V1.5 — hub-side summarizer + moderator.
V2 — on-chain agent identity (PDA).
